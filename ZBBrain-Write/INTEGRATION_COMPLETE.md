## 已知限制

1. **PIL/Pillow依赖**：
   - 如果未安装Pillow，图片压缩将被跳过
   - 建议运行 `pip install Pillow>=10.0.0`

2. **主题色彩定制**：
   - 当前主题色彩是预设的
   - 如需自定义，需要修改代码中的 `theme_colors` 字典

3. **AI主题一致性**：
   - AI生成结果可能不完全符合主题规范
   - 依赖AI模型对提示词的理解能力
