## 项目简介

ZBBrain-Write 是专为 **EPC总承包** 领域打造的微信公众号自动化内容生产系统。通过 **"搜狗爬虫 → 智谱AI分析 → 总包大脑回答 → 微信发布"** 的全自动化流程，实现从行业资讯到专业文章的智能转化，大幅提升内容生产效率。

**一句话概括**: 让AI自动发现行业热点、生成专业内容、发布到微信公众号。

## 创新亮点

### 1. JTBD理论驱动的问题生成

采用 **Jobs-To-Be-Done（焦糖布丁）理论** 分析用户真实需求，从行业资讯中提炼出从业者真正关心的场景化问题。

```
传统方式: "EPC总承包有哪些注意事项？"
JTBD方式: "在EPC总承包项目中，如何有效管理设计变更引起的索赔，
         确保在责任明确的前提下，既不损害公司利益，又能合理补偿业主损失？"
```

**核心原则**:
- 场景优先：投标、设计、施工、结算等具体场景
- 真实问题：反映实际工作中的真实困惑
- 清晰准确：问题描述详细具体（50字以上）

### 2. 混合模型成本优化（v4.1）

| 任务类型 | 使用模型 | 成本 |
|----------|----------|------|
| 问题生成 | glm-4-flash | **免费** |
| 标题生成 | glm-4-flash | **免费** |
| 图片提示词 | glm-4-flash | **免费** |
| 文章分析 | glm-4-flash | **免费** |
| 内容结构化 | glm-4-flash | **免费** |
| HTML排版 | glm-4-plus | 收费 (¥0.05/千tokens) |

**成本策略**: 83%的AI任务使用免费模型，仅HTML排版保留付费模型保证排版质量。单篇成本约¥0.8，月度成本约¥192。

### 3. 10万+爆款标题生成

基于自媒体标题心理学，集成 **十大爆款标题公式**:

| 公式 | 示例 |
|------|------|
| 悬念式 | EPC设计变更索赔90%人都做错这3点很关键建议必看 |
| 数字式 | 掌握这4招让EPC项目利润直接提升30%不是梦 |
| 痛点式 | 别让设计变更毁了你的EPC项目 |
| 警告式 | EPC合同这3个条款不看清后果真的很严重 |
| 揭秘式 | 从业10年揭秘EPC总承包5个隐藏利润点 |

**标题规范**: 28-30字、包含1-2个数字、结尾建议收藏/必看

## 常见问题

### Q1: 如何获取智谱AI的API Key？
访问 [智谱AI开放平台](https://open.bigmodel.cn/)，注册后在控制台获取。

### Q2: 微信公众号IP白名单设置？
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「设置与开发」→「基本配置」
3. 将运行脚本的服务器IP添加到白名单

### Q3: 总包大脑登录状态保持？
首次运行需扫码登录，之后会自动保存登录状态到 `browser_data` 目录。

### Q4: 回答内容过短怎么办？
系统会自动重试最多5次，确保回答≥500字符。如仍不足会发送失败通知。

### Q5: 如何自定义文章主题？
修改 `config.ini` 中的 `默认主题` 配置项，可选7种主题风格。
