## 问题概述

### 问题1: 总包大脑登录状态持久化失败

**用户报告**: 用户登录后，下次运行仍然要求用户扫码登录

**影响**: 无法实现无人值守运行，用户体验极差

### 问题2: 打开总包大脑时打开多个浏览器窗口

**用户报告**: 运行时出现多个浏览器窗口

**影响**: 资源浪费，可能导致程序异常

## 根本原因分析

### 问题1 根本原因: Cookie 域名污染

```
┌─────────────────────────────────────────────────────────────┐
│                    问题流程 (v3.6.8)                          │
├─────────────────────────────────────────────────────────────┤
│ 1. 用户首次扫码登录 metaso.cn                                │
│    └─ 登录成功，保存 cookies.json (包含 metaso.cn cookies)    │
│                                                              │
│ 2. 定时任务启动 - SogouWeChatScraper                         │
│    ├─ 创建浏览器，加载 cookies.json (包含 metaso.cn cookies)  │
│    ├─ 访问 sogou.com，获取 sogou.com cookies                  │
│    └─ browser.close(save_cookies=True)                       │
│       └─ 保存所有 cookies (sogou.com + metaso.cn 混合)        │
│           ⚠️ 如果此时没有 metaso.cn cookies，会覆盖原文件！    │
│                                                              │
│ 3. 下次运行 - MetasoAutomation                               │
│    ├─ 创建浏览器，加载 cookies.json                           │
│    ├─ 发现只有 sogou.com cookies，没有有效登录状态            │
│    └─ 要求用户重新扫码登录 ❌                                 │
└─────────────────────────────────────────────────────────────┘
```

**关键代码问题**:

```python
# SogouWeChatScraper.scrape() - 第 4754 行
# 问题: 没有阻止保存 cookies
await browser.close()  # 默认 save_cookies=True，会保存 sogou.com cookies！
```

```python
# StagehandBrowser._save_cookies() - 第 4197-4204 行
# 问题: 当没有 metaso.cn cookies 时，仍然保存所有 cookies
if not metaso_cookies:
    # 仍然保存所有 cookies 作为备份 ← 这会覆盖原文件！

        json.dump(all_cookies, f)

```

### 问题2 根本原因: 浏览器关闭无超时保护

```python

# 问题: 没有超时保护，可能卡住

    await self.context.close()  # 可能卡住
if self.browser:
    await self.browser.close()  # 可能卡住
if self.playwright:
    await self.playwright.stop()  # 可能卡住
```

**影响**:
- 旧浏览器未完全关闭
- 新浏览器启动时出现多窗口
- 资源泄漏

## 修复方案

### 修复1: Cookie 域名过滤

**新增参数**: `cookie_domain_filter`

```python
async def close(self, save_cookies: bool = True, cookie_domain_filter: str = None) -> None:
    """
    关闭浏览器

    Args:
        save_cookies: 是否在关闭前保存浏览器状态
        cookie_domain_filter: 【v3.6.9新增】只保存指定域名的 cookies
            - None: 保存所有 cookies (旧行为，不推荐)
            - "metaso.cn": 只保存 metaso.cn 域名的 cookies (推荐)
    """
```

**新增方法**: `_save_storage_state_for_domain()`

```python
async def _save_storage_state_for_domain(self, domain_filter: str) -> bool:
    """只保存指定域名的浏览器状态"""
    all_cookies = await self.context.cookies()

    # 只保存指定域名的 cookies
    filtered_cookies = [
        c for c in all_cookies
        if domain_filter in c.get('domain', '')
    ]

    if not filtered_cookies:
        self.logger.warning(f"当前没有 {domain_filter} 域名的 cookies")
        return False

    # 保存过滤后的 cookies
    with open(cookies_file, 'w') as f:
        json.dump(filtered_cookies, f)
    return True
```

### 修复2: 超时保护

```python
async def close(self, ...):
    close_timeout = 30  # 30秒超时

    async def close_context():
        if self.context:
            await self.context.close()

    # 超时保护
    try:
        await asyncio.wait_for(close_context(), timeout=close_timeout)
    except asyncio.TimeoutError:
        self.logger.warning(f"关闭 context 超时，强制继续")

    # ... 同样处理 browser 和 playwright

    finally:
        # 强制清理引用
        self.context = None
        self.browser = None
        self.playwright = None
        self.page = None
```

### 修复3: 正确的调用方式

```python
# SogouWeChatScraper - 不保存任何 cookies
# 第 4754 行
await browser.close(save_cookies=False, cookie_domain_filter=None)

# MetasoAutomation.close() - 只保存 metaso.cn cookies
# 第 6001 行
await self.browser.close(save_cookies=True, cookie_domain_filter="metaso.cn")

# MetasoAutomation._restart_browser() - 只保存 metaso.cn cookies
# 第 6036 行
await self.browser.close(save_cookies=True, cookie_domain_filter="metaso.cn")
```

## 经验教训

### 1. Cookie 隔离原则

**问题**: 多个服务共享同一个 cookies 文件
**解决**: 使用域名过滤或独立的 cookies 文件

```python
# 推荐: 域名过滤
await browser.close(save_cookies=True, cookie_domain_filter="metaso.cn")

# 或者: 独立文件
cookies_file = Path("browser_data/metaso_cookies.json")  # 只用于 metaso
```

### 2. 资源清理原则

**问题**: 异步关闭操作可能卡住
**解决**: 超时保护 + finally 块强制清理

```python
try:
    await asyncio.wait_for(close_op(), timeout=30)
except asyncio.TimeoutError:
    logger.warning("关闭超时，强制继续")
finally:
    # 强制清理引用
    self.resource = None
```

### 3. 显式参数原则

**问题**: 默认行为可能不符合预期
**解决**: 显式指定所有关键参数

```python
# 不好: 依赖默认值
await browser.close()

# 好: 显式指定
await browser.close(
    save_cookies=True,
    cookie_domain_filter="metaso.cn"
)
```

### 4. 调试技巧

**问题**: Cookie 污染难以发现
**解决**: 检查 cookies.json 内容

```bash
# 检查 cookies 域名分布
cat browser_data/cookies.json | grep '"domain"' | sort | uniq -c
```
